## Day 01 — 색상 팔레트 생성기
 
### 목표
랜덤 색상 5개를 생성하고, 클릭하면 HEX 코드를 클립보드에 복사하는 도구.
 
### UI 구성
- 상단: "Generate" 버튼
- 중앙: 색상 블록 5개 (각 블록 안에 HEX 텍스트)
- 클릭 시: "Copied!" 토스트 메시지 or 블록 내 텍스트 변경
### 핵심 개념
- `Math.random()`으로 HEX 색상 생성
- `Clipboard API` (`navigator.clipboard.writeText`)
- CSS transition으로 복사 피드백
### 단계별 구현
 
**Step 1. HEX 색상 생성 함수**
```js
function randomHex() {
  return '#' + Math.floor(Math.random() * 0xFFFFFF)
    .toString(16).padStart(6, '0');
}
```
 
**Step 2. 팔레트 렌더링**
```js
function renderPalette() {
  container.innerHTML = '';
  for (let i = 0; i < 5; i++) {
    const color = randomHex();
    const block = document.createElement('div');
    block.className = 'color-block';
    block.style.background = color;
    block.textContent = color;
    block.addEventListener('click', () => copyColor(block, color));
    container.appendChild(block);
  }
}
```
 
**Step 3. 클립보드 복사 + 피드백**
```js
async function copyColor(el, hex) {
  await navigator.clipboard.writeText(hex);
  el.textContent = 'Copied!';
  setTimeout(() => el.textContent = hex, 1500);
}
```
 
**Step 4. Space bar로 재생성**
```js
document.addEventListener('keydown', e => {
  if (e.code === 'Space') renderPalette();
});
```
